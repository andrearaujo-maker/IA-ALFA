# 🤖 BOT SINAIS IA 2025

## 🚀 Funcionalidades
- Previsão automática de "Grande" 🟠 ou "Pequeno" 🔵
- Atualização a cada 5 segundos
- Acesso individual com código único
- Rodando direto no Render

## ⚙️ Instalação no Render
1. Envie esta pasta para um repositório no GitHub (nome: bot-sinais-ia)
2. No Render, clique em "New Web Service"
3. Conecte seu GitHub e selecione o repositório
4. Configure:
   - **Start Command:** `bash start.sh`
   - **Environment:** Python 3
5. Deploy e aguarde o Render iniciar

## 👤 Códigos de Acesso
